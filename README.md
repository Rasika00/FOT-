# Faculty of Technology - Rajarata University

## About This Project

This project is a modern, high-tech, and highly interactive web portal designed for the **Faculty of Technology at Rajarata University of Sri Lanka (RJT)**. 

Built with an immersive "cyber/tech" aesthetic, the site aims to provide a premium, dynamic, and engaging user experience for prospective students, current undergraduates, and faculty members. It leverages smooth scroll-snapping mechanics, custom-designed SVGs, and interactive geometric elements to showcase the cutting-edge nature of the faculty.

## Key Features

- **Immersive Tech Aesthetics:** Dark mode by default with vibrant cyan, blue, and purple neon glow effects to represent the digital era.
- **Dynamic Scroll Navigation:** Full-viewport scroll snapping for seamless, presentation-style transitions between major faculty sections (Welcome, Degrees, Hub, Location, Highlights).
- **Interactive Degree Showcases:** Dedicated, animated panels for the core pillars of the faculty, complete with interactive curriculum browsers and career target projections.
- **Real-Time Map Integration:** Embedded Google Maps showcasing the precise location of the Mihintale campus.
- **Cinematic Animations:** Powered by `framer-motion` for fluid component mounting, active states, and a dedicated, animated full-screen initializing loader.
- **Responsive Layout:** Carefully optimized for both desktop displays and mobile interfaces using Tailwind CSS.

## Technology Stack

- **Framework:** React 19 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS v4 (with custom grid and glow utilities)
- **Animations:** Framer Motion (`motion/react`)
- **Icons:** Lucide React

## Getting Started

To run this project locally on your machine:

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Navigate to the project directory:**
   ```bash
   cd faculty-of-technology---rajarata-university
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the app:**
   Open the port specified in your terminal (usually `http://localhost:5173` or `http://localhost:3000`) in your browser.
